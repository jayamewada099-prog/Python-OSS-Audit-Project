### VITyarthi Capstone Project - OSS NGMC Course

## Project Description
This project is an in-depth audit of **Python**, an open-source programming language managed by the Python Software Foundation (PSF). It covers the philosophy, licensing (PSF License), and its vital role in the Linux ecosystem.

2. **Scripts**: 5 Bash scripts for Linux system and package auditing.
3. **Audit Target**: Python 3.x
